Concurrent Tasks

A simple task runner which will run tasks in parallel while maintaining limits.

Where can I use this?

ConcurrentTasks can be used virtually anywhere. It has ZERO dependencies and works like a charm when plugged into any JavaScript environment or flavour.

  • Frontend Frameworks (React, Vue, Angular, etc)
  • Backend Frameworks (Express, Hapi, Koa, etc)
  • NPM module
  • Node CLI application
Getting Started
Using NPM 
npm i concurrent-tasks
or Yarn 
yarn add concurrent-tasks
Add as a script file
Examples:
Simplest task runner:
                    
                        import TaskRunner from 'concurrent-tasks'

                        const runner = new TaskRunner();
                        const myTask = done => {
                            setTimeout(() => {
                                done();
                            }, 1000);
                        };

                        runner.add(myTask);
