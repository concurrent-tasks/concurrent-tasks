Concurrent Tasks

A simple task runner which will run tasks in parallel while maintaining concurrency limits.

Simple task runner source: 
                    
                        import TaskRunner from 'concurrent-tasks'

                        const runner = new TaskRunner();
                        const myTask = done => {
                            setTimeout(() => {
                                done();
                            }, 1000);
                        };

                        runner.add(myTask);
